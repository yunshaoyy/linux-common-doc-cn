1.首先安装flatpak
https://flatpak.org/setup

2.再运行
flatpak install flathub org.winehq.Wine
flatpak run org.winehq.Wine winecfg

3.中文显示问题可以下载 https://drive.massgrave.dev/zh-cn_windows_10_enterprise_ltsc_2021_x64_dvd_033b7312.iso 后解包，然后复制 windows/fonts 到 ~/.var/app/org.winehq.Wine/data/wine/drive_c/windows/Fonts/
